# 🎓 Student Attendance Management System

## 📌 Overview

The **Student Attendance Management System** is a web-based application developed using PHP to efficiently manage and track student attendance.

The system provides role-based access through **Admin, Teacher, and Student panels**, enabling structured data management, attendance monitoring, and reporting within an educational environment.

---

## 🚀 Key Features

* 🔐 **Role-Based Authentication** (Admin / Teacher / Student)
* 📊 Centralized attendance tracking system
* 🧑‍🏫 Subject-wise attendance management for teachers
* 👨‍🎓 Student access to personal attendance reports
* 🗂️ Organized storage of student and teacher records
* ⚡ Simple and user-friendly interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** Apache (XAMPP/WAMP)

---

## 🧩 Modules

### 👨‍💼 Admin Panel

* Create and manage users (students & teachers)
* View complete system data
* Manage classes, sessions, and records

### 👨‍🏫 Teacher Panel

* Mark and manage student attendance
* Filter data based on subject/class
* Track attendance records

### 👨‍🎓 Student Panel

* View attendance records
* Access attendance reports

---

## ⚙️ Installation & Setup

1. Install XAMPP / WAMP
2. Move the project folder to:

   ```
   htdocs/
   ```
3. Start Apache & MySQL
4. Import the database file into phpMyAdmin
5. Run the project in browser:

   ```
   http://localhost/student-attendance-system
   ```
